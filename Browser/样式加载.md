## `<style>` VS `<link>` 加载

|            | style                                        | link                               |
| ---------- | -------------------------------------------- | ---------------------------------- |
| DOM 树解析 | 不阻塞                                       | 不阻塞                             |
| DOM 树渲染 | 不阻塞                                       | 阻塞                               |
| 样式渲染   | 使用 HTML 进行异步解析，一边解析一边渲染页面 | 需要等到样式加载完毕后才会渲染页面 |
| 页面感知   | 页面闪屏                                     | 等待一会后展示页面                 |
