# HTML 语法

在元素上以 `data-*` 开头的属性就是自定义数据属性，例如：

```html
<div data-tip="tips" data-index-number="12314" />
```

 `*` 的设置需要遵循 [xml 名称生产规则](https://www.w3.org/TR/REC-xml/#NT-Name)：

- 不能以 `xml` 开头；
- 不能包含分号；
- 不能包含 [A-Z]。
